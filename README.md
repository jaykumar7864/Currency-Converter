# Currency-Converte
A web-based application that allows users to convert one currency to another in real time using the Latest Currency API. Built with HTML, CSS, and JavaScript, the project dynamically fetches live exchange rates and displays the converted amount instantly. It also shows country flags for better user experience.

# Features

 Real-time exchange rate conversion

 Automatic country flag display

 Simple, clean, and responsive UI
 Error handling and input validation

⚡ Fast API-based conversion

# Tech Stack

HTML5 – Structure

CSS3 – Styling and layout

JavaScript (Vanilla JS) – Logic and API handling

Currency API – For fetching live exchange rates

FlagsAPI – For displaying country flags
